Prenatal vitamins, iron, BP meds, insulin. Side-effect logging tied to medication start dates.

🔒 Security Architecture

🛡️
JWT Authentication
15-minute access tokens with single-use refresh rotation. Token family tracking prevents replay attacks.
🔐
Encryption Everywhere
AES-256 at rest, TLS 1.3 in transit. Client-side encryption for sensitive health data.
📝
Audit Logging
Every data access logged: who, what, when, where. 7-year retention. Anomaly detection.
🚫
Row Level Security
Users can only access their own data. Caregiver access requires explicit patient consent.
